# プロンプト調整メモ（issue #12）

## 何を変えたか

`app/services/claude_service.rb` の `SYSTEM_PROMPT` を全面的に書き直した。

## 変更の方針

### キャラクター強化
- 「柴犬薬剤師」のキャラクターを明文化。白衣を着た柴犬で、カウンター越しにだれた感じで話しかけてくるが実は知識豊富、というペルソナを追加
- 口調のトーンを「忙しい薬局の休憩中に話しかけてくる先輩」に設定
- 説教調にならないよう「共感ベース」の指示を追加

### 薬名のユーモア強化
- 命名例を「なんとなくわかった感増強カプセル」など笑えるものに変更
- 真面目すぎる学術風の命名（「光子吸収活性化カプセル」）を脱するよう誘導

### 副作用の具体化
- 「わかる…」と思えるリアルな副作用の例を明示
  - 「飲み会でこの話をしたくなる」
  - 「関連本を3冊ポチりたくなる」

### 評価を緩和
- 旧：正確 or あいまいの二択
- 新：正確 / 惜しい / ぼんやり の3段階で、どれも「あなたはよくやった」温度感を維持
- 厳格な採点をやめて受容から入るスタンスに変更
