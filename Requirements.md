# 深夜薬局バー / Deep Night Pharmacy Bar

> 昼は薬局、夜はバーになる謎の店。マスターはサングラスの柴犬薬剤師。

---

## コンセプト

「ちょっとやってみるか」と思わせるおふざけWebアプリ。
難しい言葉は使わない。でも答えられない。調べたくなる。

### お題の3行構造
1. **固有名詞**（名前があるとかっこいい、調べられる）
2. **ルール説明**（自分で追える、1〜2行）
3. **問い**（一言）

---

## 機能

### 機能① 今夜の肴
- お題がランダムに出題される
- 「思ったこと」を自由入力
- 柴犬薬剤師が**服薬指導箋**としてAIで返答

#### テーマ
- 未解決でも気軽に答えるんだ、今思ったことをアウトプットするんだ
- 正解・不正解は関係ない。**思ったことを言うだけでいい**
- 答えは絶対に言わない。でもキーワードはちゃんと使う

#### 服薬指導箋フォーマット

```
┌─────────────────────────────┐
　　　　服薬指導箋
　　　　担当：柴犬薬剤師 🐕
├─────────────────────────────┤
　お悩み
　　{お題の名前}への好奇心
　　（お題のルールと問いをそのまま）

　症状
　　{ユーザーの回答を要約したもの}

　処方
　　{架空の薬名} {用量}
　　{効能を一言}
　　{用法}

　注意事項・副作用
　　{ユーモアのひとこと}
├─────────────────────────────┤
　お大事にどうぞ。🐕
└─────────────────────────────┘
```

#### 架空の薬名のルール
ユーザーの回答の本質 + 作用 + 剤形（錠/液/散）でAIが毎回生成

| ユーザーの回答 | 薬名の例 |
|--------------|---------|
| 宇宙の摂理っぽい | 収束感錠 100mg |
| 眠いからじゃない？ | 睡眠誘発性回避液 |
| なんかループしてる | 無限軌道緩和散 |

#### 服薬指導箋サンプル

```
┌─────────────────────────────┐
　　　　服薬指導箋
　　　　担当：柴犬薬剤師 🐕
├─────────────────────────────┤
　お悩み
　　コラッツ予想への好奇心
　　偶数なら÷2、奇数なら×3+1を繰り返すと
　　必ず1になる。なぜ？

　症状
　　宇宙の摂理との接続を試みている

　処方
　　収束感錠 100mg
　　1に帰る必然性への納得感を一時的に高める
　　就寝前1錠服用

　注意事項・副作用
　　素面で数学者に話さないこと
├─────────────────────────────┤
　お大事にどうぞ。🐕
└─────────────────────────────┘
```

### 機能② 今日の副作用レポート
- 今日やったことを自由入力
- **効能・効果 / 副作用 / 今夜の処方（酒）** がAIで生成される

## 技術構成（予定）

```
deep_night_pharmacy_bar/
├── app/
│   ├── controllers/
│   │   ├── topics_controller.rb      # 今夜の肴
│   │   └── side_effects_controller.rb # 今日の副作用
│   ├── models/
│   │   └── topic.rb                  # お題データ
│   ├── services/
│   │   └── claude_service.rb         # Claude API呼び出し
│   └── views/
│       ├── topics/
│       └── side_effects/
├── db/
│   └── seeds.rb                      # お題の初期データ
└── config/
    └── routes.rb
```

### 使用技術
- Ruby on Rails 7
- Claude API / claude-haiku-4-5-20251001（返答生成）
- DBはほぼ不要（お題はseedsで管理）

### コスト試算
| 月間リクエスト数 | 月額コスト |
|--------------|----------|
| 100回（自分だけ） | 約14円 |
| 1,000回 | 約140円 |
| 10,000回 | 約1,400円 |

※ 物足りなければClaude Sonnet 4.6に切り替え（1リクエスト約0.6円）
